# ドキュメント構造テンプレート

基本的なドキュメント構造（必要に応じてカスタマイズ）

## チュートリアル形式

```markdown
# [タイトル]

## 概要
[簡潔な説明]

## 前提条件
- [必要な知識・環境]

## 手順
1. [ステップ1]
2. [ステップ2]

## 例
[コード例やスクリーンショット]

## トラブルシューティング
- 問題: [問題内容]
  解決: [解決方法]

## 参考リンク
- [関連ドキュメント]
```

## APIリファレンス形式

```markdown
# [API名]

## 概要
[簡潔な説明]

## 関数一覧

### 関数名

**説明**: [関数の説明]

**パラメータ**:
| 名前 | 型 | 必須 | 説明 |
|------|-----|------|------|
| param1 | string | はい | [説明] |

**戻り値**: [戻り値の説明]

**例**:
\```language
[コード例]
\```

**例外**:
- [例外の説明]
```

## README形式

```markdown
# [プロジェクト名]

## 概要
[プロジェクトの簡潔な説明]

## 特徴
- [特徴1]
- [特徴2]

## インストール
\```bash
[インストールコマンド]
\```

## 使い方
[基本的な使い方]

## ドキュメント
- [リンク]

## ライセンス
[ライセンス情報]
```

## 保存先

| ドキュメント種別 | 保存先 |
|-----------------|--------|
| 一般ドキュメント | `docs/` |
| プラン関連 | `ai/plans/` |
| テンプレート | `ai/templates/` |
| レビュー結果 | `ai/reviews/` |
